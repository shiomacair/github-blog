Ruby で素因数分解

Ruby で素因数分解の関数を作りました。 

def prime_factors(n)
  factors = []
  i = 2
  while i * i <= n
    while n % i == 0
      factors << i
      n /= i
    end
    i += 1
  end
  factors << n if n > 1
  factors
end

# テスト例
puts prime_factors(60).inspect  # => [2, 2, 3, 5]
puts prime_factors(97).inspect  # => [97] (素数)
puts prime_factors(1).inspect   # => [] (1は素因数なし)

先に2で割るようにした方が、高速化できるかもしれません。
計算量なども考えてみたいです。
